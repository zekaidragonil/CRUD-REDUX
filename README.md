# CRUD de Clientes - React, Redux & Axios

Este es un proyecto de práctica desarrollado para dominar la gestión de estado global y el consumo de APIs REST en entornos modernos de React. Se enfoca en la implementación de un **CRUD** (Create, Read, Update, Delete) robusto y escalable.

## 🚀 Tecnologías Principales

* **Frontend:** [React 18](https://reactjs.org/) con [Vite](https://vitejs.dev/) para un desarrollo ultrarrápido.
* **Estado Global:** [Redux Toolkit (RTK)](https://redux-toolkit.js.org/) para una lógica de estado predecible y simplificada.
* **Peticiones HTTP:** [Axios](https://axios-http.com/) para la comunicación con servicios REST.
* **Enrutado:** [React Router DOM v6](https://reactrouter.com/) para la navegación SPA.
* **UI & Notificaciones:** [SweetAlert2](https://sweetalert2.github.io/) para una experiencia de usuario interactiva y elegante.
